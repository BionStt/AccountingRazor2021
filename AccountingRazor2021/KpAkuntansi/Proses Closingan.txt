Proses closingan bulanan dan tahunan dalam akuntansi melibatkan langkah-langkah untuk memastikan bahwa semua transaksi yang terjadi dalam periode tertentu telah dicatat dengan benar, laporan keuangan disiapkan, dan saldo akun ditutup dan dibuka untuk periode berikutnya. Berikut adalah contoh proses untuk closingan bulanan dan tahunan:

Contoh Proses Closingan Bulanan
1. Persiapan Awal
Pastikan semua transaksi bulan tersebut sudah dicatat.
Verifikasi bahwa semua jurnal penyesuaian telah dilakukan.
Rekonsiliasi akun kas dan bank.
Pastikan semua akun pendapatan dan beban sudah sesuai.
2. Proses Penutupan Akun Pendapatan dan Beban
sql
Salin kode
-- Menghitung total pendapatan bulan tersebut
INSERT INTO SummaryIncome (Month, Year, TotalIncome)
SELECT MONTH(GETDATE()), YEAR(GETDATE()), SUM(Amount)
FROM Revenue
WHERE DATEPART(MONTH, Date) = MONTH(GETDATE()) AND DATEPART(YEAR, Date) = YEAR(GETDATE());

-- Menghitung total beban bulan tersebut
INSERT INTO SummaryExpense (Month, Year, TotalExpense)
SELECT MONTH(GETDATE()), YEAR(GETDATE()), SUM(Amount)
FROM Expenses
WHERE DATEPART(MONTH, Date) = MONTH(GETDATE()) AND DATEPART(YEAR, Date) = YEAR(GETDATE());
3. Transfer Laba Rugi Bulanan ke Akun Laba Rugi Ditahan
sql
Salin kode
-- Menutup akun pendapatan ke akun laba rugi bulan berjalan
INSERT INTO AccountClosure (AccountID, Amount, ClosureDate)
SELECT AccountID, -SUM(Amount), GETDATE()
FROM Revenue
WHERE DATEPART(MONTH, Date) = MONTH(GETDATE()) AND DATEPART(YEAR, Date) = YEAR(GETDATE())
GROUP BY AccountID;

-- Menutup akun beban ke akun laba rugi bulan berjalan
INSERT INTO AccountClosure (AccountID, Amount, ClosureDate)
SELECT AccountID, SUM(Amount), GETDATE()
FROM Expenses
WHERE DATEPART(MONTH, Date) = MONTH(GETDATE()) AND DATEPART(YEAR, Date) = YEAR(GETDATE())
GROUP BY AccountID;

-- Transfer laba rugi bulan berjalan ke akun laba rugi ditahan
DECLARE @monthlyProfitLoss DECIMAL(18,2);
SET @monthlyProfitLoss = (SELECT SUM(Amount) FROM AccountClosure WHERE ClosureDate = GETDATE());

INSERT INTO RetainedEarnings (Date, Amount)
VALUES (GETDATE(), @monthlyProfitLoss);
4. Pembukaan Saldo Awal untuk Bulan Berikutnya
sql
Salin kode
-- Membuka saldo awal untuk bulan berikutnya
INSERT INTO OpeningBalance (AccountID, Amount, Date)
SELECT AccountID, Balance, DATEADD(MONTH, 1, GETDATE())
FROM Accounts
WHERE AccountType IN ('Asset', 'Liability', 'Equity');
Contoh Proses Closingan Tahunan
1. Persiapan Awal
Verifikasi bahwa semua transaksi tahun tersebut sudah dicatat.
Pastikan semua jurnal penyesuaian tahunan telah dilakukan.
Rekonsiliasi akhir tahun untuk semua akun utama.
2. Penutupan Akun Pendapatan dan Beban Tahunan
sql
Salin kode
-- Menghitung total pendapatan tahun tersebut
INSERT INTO YearlySummaryIncome (Year, TotalIncome)
SELECT YEAR(GETDATE()), SUM(Amount)
FROM Revenue
WHERE DATEPART(YEAR, Date) = YEAR(GETDATE());

-- Menghitung total beban tahun tersebut
INSERT INTO YearlySummaryExpense (Year, TotalExpense)
SELECT YEAR(GETDATE()), SUM(Amount)
FROM Expenses
WHERE DATEPART(YEAR, Date) = YEAR(GETDATE());
3. Transfer Laba Rugi Tahunan ke Akun Laba Rugi Ditahan
sql
Salin kode
-- Menutup akun pendapatan ke akun laba rugi tahunan
INSERT INTO AccountClosure (AccountID, Amount, ClosureDate)
SELECT AccountID, -SUM(Amount), GETDATE()
FROM Revenue
WHERE DATEPART(YEAR, Date) = YEAR(GETDATE())
GROUP BY AccountID;

-- Menutup akun beban ke akun laba rugi tahunan
INSERT INTO AccountClosure (AccountID, Amount, ClosureDate)
SELECT AccountID, SUM(Amount), GETDATE()
FROM Expenses
WHERE DATEPART(YEAR, Date) = YEAR(GETDATE())
GROUP BY AccountID;

-- Transfer laba rugi tahunan ke akun laba rugi ditahan
DECLARE @yearlyProfitLoss DECIMAL(18,2);
SET @yearlyProfitLoss = (SELECT SUM(Amount) FROM AccountClosure WHERE ClosureDate = GETDATE());

INSERT INTO RetainedEarnings (Date, Amount)
VALUES (GETDATE(), @yearlyProfitLoss);
4. Pembukaan Saldo Awal untuk Tahun Berikutnya
sql
Salin kode
-- Membuka saldo awal untuk tahun berikutnya
INSERT INTO OpeningBalance (AccountID, Amount, Date)
SELECT AccountID, Balance, DATEADD(YEAR, 1, GETDATE())
FROM Accounts
WHERE AccountType IN ('Asset', 'Liability', 'Equity');
Form yang Dibutuhkan
Form Jurnal Penyesuaian Bulanan: Untuk mencatat semua penyesuaian bulanan.
Form Rekonsiliasi Kas dan Bank: Untuk mencocokkan saldo kas dan bank.
Form Rekonsiliasi Akun: Untuk memverifikasi semua saldo akun.
Form Laporan Laba Rugi Bulanan: Untuk menyajikan pendapatan dan beban bulan tersebut.
Form Transfer Laba Rugi ke Akun Ditahan: Untuk mencatat transfer laba rugi ke akun laba rugi ditahan.
Form Pembukaan Saldo Awal: Untuk mencatat saldo awal bulan berikutnya.
